# Cisco-Network-Lab (Router+switch):
Configured a Cisco router and switch in the Packet Tracer to create a LAN network, assign IP addresses, and verify connectivity using ping. 
This project demonstrates a basic local Area Network(LAN) setup using Cisco Packet Tracer. A router and switch were configured to allow multiple PCs to communicate within the same network.

# Tools and Technology Used:
Cisco Packet Tracer
Cisco Router (GigabitEthernet)
Switch (Layer2)
IPV4 Addressing

# Configuration steps:
Entered privileged mode and configure mode on the router
Configured router interface; Ip Address 192.168.1.1, Subnet Mask; 255.255.255.0
Enabled interface using "no shutdown" 
Assigned Static Ip addresses to all PCs 
Configured default gateway for each PC

# Verification:
Used "show ip interface brief" to verify router status
Tested connectivity using "ping" between PCs and router
Confirmed all device successfully communicate

# Result:
All devices in the network were able to communicate successfully.
